什么是a标签

（或称锚元素）可以创建通向其他网页、文件、同一页面内的位置、电子邮件地址或任何其他 URL 的超链接。

常见单词

href:hyper reference //超级引用,超链接target //目标

属性

href 包含超链接指向的 URL 或 URL 片段,当值为#xxx.会指定到达IDxxx元素

href="http://google.com" //使用http协议指向URL
href="https://google.com" //使用https协议指向URL
href="//google.com" //无协议指向URL,会依次使用http和https

href="/a/b/index.html" //指向根目录下的a/b/index.html
href="a/b/index.html" //指向a/b/index.html

href="javascript:alert('hello world!')"  //使用伪协议,可以制作无操作的超链接
href="emilto:1234@foxmail.com"  //指向E_mailo

id href="#user"  //跳转到指定id元素

target  属性指定在何处显示链接的资源。 取值为标签（tab），窗口（window） 

target=_blank  //新标签页打开
target=_top //在当前页面顶层打开
target=_parent //在父框架下打开
target=_self //在当前标签下打开

download 此属性指示浏览器下载 URL 而不是导航到它rel=noopener 该属性指定了目标对象到链接对象的关系。该值是空格分隔的列表类型值。

什么是table

table 就是表格

常见单词

table //表格thead //表头tbody //表身tfoot //表尾td : table descript //单元格描述th :able head //表头tr :table row 行

table 可以干什么

将数据填写到表格容器里,展示

table的书写规范是什么

不用管有几行单元格.都要使用table标签包围,分别使用thead,tbody,tfoot.

table 常见的样式

table-layout :auto //自动分配宽度border-spacing: //表示表格之间的间隙border-collapse:collapse //决定单元格之间是合并

代码

    <table>
        <thead>
            <tr>
                <th>单词</th>
                <th>翻译</th>
            </tr>
            <!-- table row -->
            <!-- table head -->
        </thead>
        <tbody>
            <tr>
                <td>hyper</td>
                <td>超级</td>
              <!-- table descript -->
            </tr>
        </tbody>
        <tfoot></tfoot>
    </table>

什么是img

发出get请求,显示图片

常见属性

src //指向图片位置,alt //图片描述width //宽度 ;max-width:100% //响应式宽度
